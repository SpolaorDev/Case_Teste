Bem-vindo ao Site de Testes

O sete cases de teste

Positivo

Testa entradas válidas e espera que o sistema funcione corretamente.

Negativo

Testa entradas inválidas para verificar como o sistema lida com erros.

Limite

Avalia o comportamento do sistema nas extremidades dos valores aceitáveis.

Exceção

Testa situações raras ou inesperadas para ver como o sistema responde.

Regressão

Garante que novas mudanças não afetem funcionalidades existentes.

Integração

Verifica a interação entre diferentes módulos ou componentes do sistema.

Desempenho

Avalia a eficiência do sistema sob carga, incluindo tempo de resposta e uso de recursos.
